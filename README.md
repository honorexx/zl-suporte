# Sistema de Cadastro de Equipe de Suporte

Este projeto foi desenvolvido utilizando PHP, MySQL, HTML e CSS.

## Funcionalidades

- Cadastro de membros da equipe
- Listagem de registros
- Edição de dados
- Exclusão de registros

## Tecnologias utilizadas

- PHP
- MySQL
- HTML
- CSS
- JAVASCRIPT
